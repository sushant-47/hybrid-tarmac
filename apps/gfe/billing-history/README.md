
### Billing History
- Build a responsive payment history section that allows users to view and download their past invoice details

- https://www.greatfrontend.com/projects/challenges/billing-history-section

### About
- Uses Zoneless change detection provided in Angular v20.3.13 using `provideZonelessChangeDetection` in `app.config`.
- CSS Theming for branding and light/dark mode.

#### CSS
- Adding a color to the app:
    1. Determine the usage of color (ex: primary, secondary, error, success, foreground, background).
    2. Add the color to color palette in `src/app/styles/colors/color.scss`.
    3. Create a color variable in `src/app/styles/colors/variables.scss`.
    4. Use the created SCSS color variable to create a CSS variable in `src/app/styles/reset.scss`.
